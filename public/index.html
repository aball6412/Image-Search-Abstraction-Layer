Image Search Abstraction Layer


How to Use:

1. Pass a search query into the URL to the path:


     http://localhost:3000//api/my_search_query

2. In return, our server will make an API connection with imgur and display the search results in JSON.

3. If you wish to see the 10 most recent searches go to the path:


     http://localhost:3000/api/latest/imagesearch

Example:

http://localhost:3000//api/tesla
http://localhost:3000//api/spacex

**In order to have a better view of raw JSON data use a browser plugin
